**Git** — A distributed version control system used to track changes in software development and enable collaboration among developers.

---

This project presents a detailed audit of Git, an open-source software. It covers:

* Origin and philosophy of Git
* License analysis (GPL v2)
* Ethical aspects of open source
* Linux footprint and system behavior
* Role of Git in the open-source ecosystem
* Comparison with proprietary software

---

Displays system information such as kernel version, user, uptime, and date.

Checks whether Git is installed and displays its version along with a description.

Analyzes selected directories and displays their permissions and size (optimized for Git Bash).

Reads a file, counts occurrences of a keyword, and displays matching lines.

Takes user input and generates a personalized open-source manifesto saved in a text file.
